# AGENTS_CODING.md

Coding behavior for WordPress standards (non-security).

Authority:
- CODING_RULES.md
- SECURITY_RULES.md (if security is touched)

Defaults:
- Follow CODING_RULES.md.
- Use the single most relevant skill from SKILLS_INDEX.md.

---

## Mandatory behavior

- Preserve tab indentation.
- Avoid large reformatting unless requested.
- Avoid lint-only changes mixed with logic changes unless requested.

---

## Refusals

- Refuse when CODING_RULES.md requires it and state the CR-REFUSE ID.

END OF AGENTS_CODING.
